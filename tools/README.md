# Local Wolf Tools

Put local third-party binaries here when running att-wolf:

- `WolfTL.exe`
- `UberWolf.exe`
- `WolfDec.exe`
- `wolf-runtime-pro\EditorPro.exe` and `wolf-runtime-pro\GamePro.exe` are stored with this repository as the default high-version WOLF Pro runtime.
- `wolf-runtime\Editor.exe` and `wolf-runtime\Game.exe` are stored with this repository as the free-edition fallback runtime.

`WolfTL.exe`, `UberWolf.exe`, and `WolfDec.exe` are intentionally local. You can also configure paths in `wolf-source\setting.toml` or through environment variables:

- `ATT_WOLF_WOLFTL_PATH`
- `ATT_WOLF_UBERWOLF_PATH`
- `ATT_WOLF_WOLFDEC_PATH`
- `ATT_WOLF_RUNTIME_DIR`

The bundled `wolf-runtime-pro` is WOLF RPG Editor Pro 3.595.2025.503. The bundled fallback `wolf-runtime` is WOLF RPG Editor v3.703 mini.
